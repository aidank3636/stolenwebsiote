Current state of my workflow is written out in [config repo](https://github.com/nikitavoloboev/config).

### Websites

- [React](https://react.dev) to render, [Jazz](https://jazz.tools)/[Reatom](https://v1000.reatom.dev) for state
- Deployed on [Vercel](https://vercel.com)
- Also use [Blade](https://blade.im) with [Ronin](https://ronin.co) (great ORM)
- [Panda CSS](https://panda-css.com) or [Tailwind](https://tailwindcss.com) for styling
- [Better Auth](https://www.better-auth.com) for auth

### HTTP

- TS: [Hono](https://hono.dev)/[Elysia](https://elysiajs.com)/[Rivet](https://rivet.gg)/[Encore](https://encore.dev) depending on task
- Deployed usually on [Railway](https://railway.com), Cloudflare or GCP

### Scripts / programs

- Prefer [Bun](https://bun.sh) with [Optique](https://github.com/dahlia/optique) for CLI

### iOS

- [Expo](https://expo.dev) + Expo plugins
- [Unistyles](https://unistyl.es) for styles
- [Jazz](https://jazz.tools)/[Reatom](https://v1000.reatom.dev) for state
- [Better Auth](https://www.better-auth.com) for auth
